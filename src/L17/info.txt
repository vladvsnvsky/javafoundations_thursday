Un tip de data reprezinta dimensiunea si tipul unei valori
memorata intr-o o variabila.

Tipurile de data sunt de 2 feluri:
1. Primitive
2. Complex

Java este un libaj de programare static-typed => inseamna ca tipurile de
data ale variabilelor sunt cunoscute la momentul compilarii

Compilarea este procesul care transforma codul sursa in cod binar (bytecode).

1. Tipurile de data PRIMITIVE - sunt in principiu blocurile de baza ce pot
memora doar valori simple direct in memorie:
boolean
char
byte
short
int
long
float
double

2. Tipurile de data NEPRIMITIVE/COMPLEX
- String  // este un sir de variabile de tip char
- Array //sir de variabile de acelasi tip
- Orice alta clasa pe care o cream poate fi un tip de data complex

O lista este un tip de data complex
Listele implementeaza in general interfara List: https://docs.oracle.com/en/java/javase/11/docs/api/java.base/java/util/List.html

In lista putem adauga si elimina elemete
Nu are o dimensiune constanta, spre deosebire de Array
